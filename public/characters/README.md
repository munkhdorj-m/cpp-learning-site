# AI-Generated Characters

Drop your AI-generated character images here with transparent backgrounds (PNG or WebP).

## Files to add:

| File               | Placement                                | Recommended size |
| ------------------ | ---------------------------------------- | ---------------- |
| `coder-left.png`   | Floating on the left side of the hero    | ~300×400px       |
| `coder-right.png`  | Floating on the right side of the hero   | ~300×400px       |
| `coder-center.png` | Small floating character near the bottom | ~150×200px       |

## Tips for generation

Use an AI image generator (Midjourney, DALL·E, etc.) with a prompt like:

> "Cute cartoon character of a kid programmer, pixel art style or flat illustration, transparent background, excited expression, holding a laptop or typing on a keyboard, vibrant colors, gaming aesthetic, suitable for 7th-8th grade students"

Remove the background before placing here (use remove.bg or similar).

## Fallback

If images are missing, the component uses inline SVG placeholders — nothing breaks.
